# house-price-prediction-python
This project builds a linear regression model to predict house prices based on various features such as number of rooms, location, size, and other relevant factors.
# Features
Loaded and preprocessed housing dataset
Handled missing values and cleaned data
Trained a Linear Regression model
Predicted house prices based on input features
Evaluated model performance
# Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
# Description
This project is part of the QSkill Internship Beginner tasks.
It demonstrates basic machine learning concepts including data preprocessing, model training, and prediction using linear regression.
